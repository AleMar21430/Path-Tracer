## Controls:
```
W/S : Forwards/Backwards
A/D : Left/Right
E/Q : Up/Down

R   : Reread and Recompile Shader
V   : View Mode Cycle (Path-traced / ZDepth / Ambient Occlusion)
C   : Reset Camera Position
P   : Toggle Pause
ESC : Exit

Right-Click + Move : Look Around
Scroll-Wheel       : Increase Movement Sensitivity
```
